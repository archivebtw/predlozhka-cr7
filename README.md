# ПРЕДЛОЖКА CR7 — версия с Supabase

Теперь игры хранятся в общей облачной базе. После добавления из `admin.html` карточка появляется на `index.html` сразу, без замены файлов в GitHub.

## Файлы

- `index.html` — публичный каталог.
- `admin.html` — вход владельца, добавление, редактирование и удаление игр.
- `config.js` — Project URL и публичный ключ Supabase.
- `supabase_setup.sql` — создание таблиц, защиты RLS и Realtime.

`games.json` больше не нужен.

## Настройка Supabase

### 1. Создай проект

Создай новый проект в Supabase.

### 2. Создай таблицы и защиту

Открой **SQL Editor**, вставь всё содержимое `supabase_setup.sql` и нажми **Run**.

### 3. Создай единственного администратора

1. Открой **Authentication → Users**.
2. Создай пользователя с твоим email и надёжным паролем.
3. Скопируй UUID пользователя.
4. В SQL Editor выполни:

```sql
insert into public.site_admins (user_id)
values ('ВСТАВЬ-СЮДА-UUID');
```

5. В настройках Authentication отключи публичную регистрацию новых пользователей, чтобы аккаунты нельзя было создавать снаружи.

### 4. Заполни config.js

В панели Supabase открой настройки API и скопируй:

- Project URL;
- Publishable key или legacy `anon` key.

Вставь их в `config.js`:

```js
window.CR7_CONFIG = Object.freeze({
  supabaseUrl: 'https://твой-проект.supabase.co',
  supabasePublishableKey: 'твой-публичный-ключ'
});
```

Не вставляй `service_role` key — он секретный и не должен находиться в браузере.

## Публикация на GitHub Pages

Загрузи в репозиторий:

- `index.html`
- `admin.html`
- `config.js`

`supabase_setup.sql` и `README.md` можно не публиковать, но удобно хранить их в репозитории.

После этого:

- публичный адрес сайта открывает `index.html`;
- панель владельца находится по адресу `.../admin.html`;
- даже если кто-то узнает адрес админки, записывать данные без твоего аккаунта ему не позволит RLS.

## Проверка

1. Открой `admin.html`.
2. Войди своим email и паролем.
3. Добавь тестовую игру.
4. Открой `index.html` в другой вкладке.
5. Новая карточка должна появиться автоматически.

Если Realtime не подключился, обновление страницы всё равно загрузит актуальный список из Supabase.
