# Как безопасно загрузить модульное обновление

1. Скачайте резервную копию текущего `config.js` из GitHub.
2. Распакуйте архив.
3. Скопируйте текущий `config.js` в корень папки рядом с `index.html`.
4. В GitHub удалите старые `index.html` и `admin.html` либо замените их новыми.
5. Загрузите целиком папку `assets`.
6. Загрузите новые `index.html`, `admin.html` и `README.md`.
7. **Не заменяйте рабочий `config.js`.**
8. Папку `supabase` можно загрузить как резервную копию, но повторно запускать SQL не требуется.
9. Дождитесь зелёной галочки в `Actions`.
10. Откройте сайт и нажмите `Ctrl + F5`.

После проверки следующим этапом лучше отдельно доработать `twitch-theme.css` и мобильную версию, не трогая Supabase-логику.
