Управление каталогом

Игры сохраняются прямо в Supabase и сразу появляются на публичном сайте. Изменения разрешены только аккаунту, добавленному в список администраторов.

Supabase не настроен

Открой config.js и вставь Project URL и публичный publishable/anon key.

Вход владельца

Используй email и пароль пользователя, которого ты создал в Supabase Auth.

Публикации отправляются напрямую в базу. Перезагружать GitHub Pages после добавления игры больше не нужно.

Добавить игру

Заполни карточку. Обязательные поля отмечены звёздочкой.

Можно оставить пустым — тогда карточка покажет название игры вместо картинки.

Меньшее число располагается выше. При одинаковом числе новая игра показывается первой.
